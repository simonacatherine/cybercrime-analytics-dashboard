# Cybercrime Analytics Dashboard

## Project Overview

This project analyzes cybercrime trends across Indian states using publicly available cybercrime and phishing datasets. The analysis focuses on identifying high-risk regions, dominant cybercrime categories, fraud trends, and crime distribution patterns through data analysis and interactive dashboards.

## Tech Stack

* Python (Pandas, Matplotlib)
* SQL (MySQL)
* Power BI
* Excel

## Objectives

* Analyze cybercrime trends across Indian states
* Identify high-risk crime categories and regional hotspots
* Examine fraud concentration and cybercrime growth patterns
* Build interactive dashboards for visual analysis

## Key Insights

* Telangana recorded the highest cybercrime activity and fraud concentration.
* Karnataka showed consistently high cybercrime activity across years.
* Fraud dominated reported cybercrime categories nationally.
* Population alone did not explain cybercrime intensity across states.
